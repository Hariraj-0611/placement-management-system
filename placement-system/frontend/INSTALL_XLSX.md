# Install XLSX Package for Excel Export

## Installation

Run this command in the frontend directory:

```bash
cd placement-system/frontend
npm install xlsx
```

OR

```bash
cd placement-system/frontend
yarn add xlsx
```

## What is XLSX?

XLSX (SheetJS) is a JavaScript library for reading and writing Excel files. It allows the frontend to export data to Excel format (.xlsx) without needing backend processing.

## Features Added

- Export all applications to Excel
- Export applications by company
- Export applications by status
- Automatic column width adjustment
- Professional Excel formatting

## File Size

- Package size: ~1.5 MB
- No additional dependencies required

## Browser Compatibility

✅ Chrome/Edge
✅ Firefox  
✅ Safari
✅ Mobile browsers

## After Installation

1. Install the package: `npm install xlsx`
2. Restart the development server: `npm run dev`
3. The export buttons will now work!
